# Telecom Customer ETL Pipeline

A beginner ETL pipeline project built using Python and Pandas.

## Features
- Extracts telecom customer data from CSV
- Cleans missing and duplicate records
- Standardizes column names
- Creates business transformations
- Categorizes customer usage patterns
- Handles ETL pipeline exceptions
- Saves transformed output data
